# Terminal Basics - Your New Friend 🐶

## Top 10 Commands

Here are the top 10 terminal commands you'll use most often:

1. **`ls`** - List files and folders in current directory
2. **`cd foldername`** - Change to a different folder
3. **`pwd`** - Show where you are (current folder path)
4. **`cat filename`** - Display contents of a file
5. **`mkdir foldername`** - Create a new folder
6. **`rm filename`** - Delete a file
7. **`cp file1 file2`** - Copy a file
8. **`mv file1 file2`** - Move or rename a file
9. **`git status`** - Check what files changed (in git projects)
10. **`npm run dev`** - Start your development server (for web projects)

## Bonus Tips

- Press `Tab` to auto-complete file/folder names
- Press `↑` (up arrow) to see previous commands
- Type `clear` to clean up the screen
- Press `Ctrl+C` to stop a running command
